## What is Wraft?

Wraft is a structured content authoring platform built on open formats — Markdown and JSON. Documents are generated from reusable templates using **Typst** or **LaTeX** rendering engines, producing pixel-perfect PDFs every time. Teams define document structures once as **Content Types**, then authors fill in the fields and Wraft handles the rest.

## Getting Started

<Steps>
  <Step title="Deploy Wraft">
    Run Wraft locally or on your server using Docker Compose. See the [self-hosting guide](/self-hosting) for full setup instructions including PostgreSQL, MinIO, and Typesense.
  </Step>
  <Step title="Authenticate">
    Sign in and obtain a JWT token, or create an API key for programmatic access. See [authentication](/api/authentication) for details.
  </Step>
  <Step title="Create a Content Type">
    Define your document structure — fields, layout, theme, and approval flow. Content Types are the blueprints for all documents in Wraft.
  </Step>
  <Step title="Generate Documents">
    Create document instances from your Content Type. Wraft renders them to PDF using Typst or LaTeX and stores them in MinIO.
  </Step>
</Steps>
